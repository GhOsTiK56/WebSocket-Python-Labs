# WebSocket Python Project

Этот проект содержит различные примеры использования WebSocket с Python, включая реализации с использованием Socket.IO, Flask, FastAPI и других фреймворков.

## Установка зависимостей

Для установки всех необходимых зависимостей выполните команду:

```bash
pip install -r requirements.txt
```

## Структура проекта

- `lab2/lab2.1-2.3/` - Простой сервер Socket.IO
- `lab2/lab2.4/` - Различные реализации серверов (AioHttp, Eventlet, Uvicorn, Werkzeug)
- `lab2/lab2.5/` - Примеры интеграции с Flask и FastAPI
- `lab2/lab5_project1/` - Полноценный проект с WebSocket функциональностью
